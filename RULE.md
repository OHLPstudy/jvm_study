# RULE
성공적인 스터디 완수를 위하여 공부 방식에 대한 기본적인 컨벤션을 지킵시다

## 스터디 방식
1. 작성되어 있는 스터디 목차와 다른 방식으로 타임라인을 풀어가도 됩니다.
   - 하지만 결론적으로 이야기하는 최종 목적엔 도달되어야 합니다.(힙영역에 대한 분석이 가능해야한다)
2. 매주 토요일 10:00 ~ 13:00 게더타운으로 스터디를 진행합니다
[게더타운](https://app.gather.town/app/5N8Mp9szfUPzfgll/OHLP_Study) 
3. 마지막 주 토요일은 오프라인 리뷰를 진행합니다
  - 리뷰 방식은 아래에 상세히 기입하겠습니다,.
4. 온라인 자율학습으로 진행하기 떄문에 꼭 같이 토요일 오전에 진행 할 필요는 없습니다. 주에 한번 정리하여 Issues 또는 Pull Request로 작성되면 됩니다.
  - 하지만 서로 질의응답을 하려면 같은 시간대에 공부하는 게 좋으니 웬만하면 지정 된 시간에 합시다
5. 어려운 부분이 있어 같이 해결하거나 공유하고 싶은 지식이 있다면 자유롭게 Issue 탭에 작성합니다.
6. 해당 RULE.md 는 누구나 수정 가능합니다.

## 규칙
1. 주에 최소 3시간은 공부해야합니다.
2. AI 사용은 자유로우나 AI을 참고한 지식은 무조건 참고사이트를 명시해야합니다
e.g) STW(Stop the World)는 JVM이 메모리를 호율적으로 관리하기 위해 GC작업 수행 시 발생하는 오버헤드를 뜻합니다.
